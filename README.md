# Multilingual Website About a Great Figurative Artist

A Flask-based multilingual website dedicated to showcasing the life, works, and impact of a renowned figurative artist. This website provides an immersive experience in multiple languages, allowing users from around the world to explore the artist's biography, gallery, news, and more.

## Features

- **Multilingual Support**: Content available in multiple languages to cater to a global audience.
- **Dynamic Content**: Information about the artist, their artworks, exhibitions, and latest news.
- **Responsive Design**: Fully responsive design for optimal viewing on all devices.
- **User-Friendly Navigation**: Intuitive and easy-to-use navigation to enhance user experience.
- **SEO Optimized**: Built with SEO best practices to improve search engine visibility.

## Installation

To run this project locally, follow these steps:

1. **Clone the repository**:
    ```bash
    git clone https://github.com/yourusername/multilingual-artist-website.git
    cd multilingual-artist-website
    ```

2. **Create a virtual environment**:
    ```bash
    python3 -m venv venv
    source venv/bin/activate   # On Windows use 'venv\Scripts\activate'
    ```

3. **Install dependencies**:
    ```bash
    pip install -r requirements.txt
    ```

4. **Run the Flask application**:
    ```bash
    flask run
    ```

5. **Access the website**: Apri il tuo browser e vai a `http://127.0.0.1:5000/`.

## Usage

Visit the homepage to learn more about the artist. Use the language switcher in the navigation bar to change the language. Explore different sections such as the biography, gallery, and news to discover more about the artist's life and works.
